React Native Learning Plan

A hands-on, 2-week plan tailored to your background — pick your track and dive in.

🖥️

Backend Developer Track

You know APIs, data, and server logic — but the UI and mobile layer is new. This plan bridges that gap fast with JS/TS fundamentals before jumping into React Native.

REST APIs ✓ Data modeling ✓ JS/React → learn first Mobile UI → new concept
📋 Before You Start — Quick Prereqs (1–2 days)

JS/ES6+ You Need

  • Arrow functions, destructuring, spread
  • Promises & async/await
  • Array methods: map, filter, reduce
  • Modules: import / export

React Basics You Need

  • JSX syntax (like HTML in JS)
  • Functional components
  • useState & useEffect hooks
  • Props vs. state

iOS Developer Track

You know UIKit/SwiftUI, navigation patterns, and Apple's HIG. This plan maps React Native concepts directly to what you already know, so you move fast.

UIKit / SwiftUI ✓ Xcode / Instruments ✓ App lifecycle ✓ JS/React → quick ramp-up
🔄 Mental Model Map — iOS → React Native

iOS Concept

  • UIView / SwiftUI View
  • UIViewController
  • UINavigationController
  • UITabBarController
  • @State / @Binding
  • URLSession
  • UserDefaults

React Native Equivalent

  • View / component
  • Screen component
  • Stack Navigator
  • Tab Navigator
  • useState / props
  • fetch() / Axios
  • AsyncStorage
